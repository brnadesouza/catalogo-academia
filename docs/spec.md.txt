# SDD - Especificação de Design de Software

# 📌 Visão Geral

O sistema consiste em uma aplicação web de catálogo de roupas fitness desenvolvida utilizando HTML, CSS, Bootstrap e JavaScript.

A aplicação utiliza uma API pública para carregamento dinâmico dos produtos.

# 🧱 Estrutura do Projeto

    └── sdd.md

## 📦 Estrutura de Dados

### Produto
- nome
- preço
- categoria
- imagem

### Categoria
- masculino
- feminino
- plus size

## 🔄 Funcionalidades

### Listagem de Produtos
Exibir produtos em formato de cards utilizando Bootstrap.

### Filtro por Categoria
Permitir filtrar produtos por categoria usando JavaScript.

### Carrinho de Compras
- Adicionar produtos
- Listar itens adicionados
- Calcular total

## 🎨 Interface
- Desenvolvida com Bootstrap
- Layout responsivo
- Uso de grid system

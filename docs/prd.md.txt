# PRD - Catálogo de Roupas de Academia

## 📌 Descrição do Projeto
Este projeto consiste em um catálogo digital de roupas de academia desenvolvido como aplicação web responsiva utilizando HTML, CSS, Bootstrap e JavaScript.

O sistema permite que usuários visualizem produtos organizados em categorias, consultem preços, imagens e adicionem itens a um carrinho de compras simples.

---

# 🎯 Objetivo

Desenvolver uma aplicação web moderna e responsiva para exibição de produtos fitness, oferecendo uma navegação simples, intuitiva e acessível em diferentes dispositivos.

---

# 👤 Público-Alvo

- Pessoas interessadas em roupas fitness
- Praticantes de academia
- Atletas
- Usuários que desejam visualizar produtos online de forma rápida

---

# ⚙️ Funcionalidades Principais

## 📋 Catálogo de Produtos
- Exibição de produtos em cards
- Nome do produto
- Imagem
- Preço
- Categoria

## 🔍 Filtro de Categorias
- Masculino
- Feminino
- Plus Size

## 🛒 Carrinho de Compras
- Adicionar produtos
- Remover produtos
- Exibir quantidade de itens
- Calcular valor total

## 📱 Responsividade
- Compatível com desktop
- Compatível com tablets
- Compatível com smartphones

## 🌐 Integração com API
- Consumo de dados da Fake Store API
- Carregamento dinâmico dos produtos
